## 6. Auditoria visual e de UX

### Hierarquia visual
⚠️ **Média**. Health score card tem destaque, mas attention callout não chama atenção suficiente. Tabela é o centro, mas está densa demais.

### Spacing
⚠️ **Inconsistente**. Cards têm p-6, tabela tem px-6 py-4, drawer tem p-6. Falta ritmo vertical consistente. Alguns componentes estão apertados (type badges, flag chips).

### Tipografia
⚠️ **Fraca**. Geist fonts não estão carregadas, então fallback para system fonts. Escala tipográfica está definida mas não é usada consistentemente. Falta contraste de peso (tudo é font-medium ou font-semibold).

### Tabela de colunas
⚠️ **Densa demais**. Muita informação por linha, pouco respiro. Hover state existe mas é sutil. Sorting funciona mas UI é básica (botões pequenos no header).

### Cards
⚠️ **Genéricos**. Health score card, dataset meta card, summary stat card parecem templates shadcn/ui. Falta personalidade, refinamento de spacing interno, hierarquia visual mais forte.

### Drawer
⚠️ **Funcional mas básico**. Slide-in funciona, backdrop blur é bom, mas conteúdo interno é denso. Histogram é pequeno (h-64), stats são apresentados de forma tabular sem refinamento.

### Empty state
❌ **Fraco**. Ícone genérico, texto simples, sem personalidade. Não convida à ação.

### Loading state
❌ **Básico**. Spinner genérico com border-t-teal. Sem skeleton, sem progressão, sem feedback rico.

### Error state
✅ **OK**. Ícone de warning, mensagem, botão de retry. Funcional mas poderia ser mais refinado.

### Consistência
✅ **Boa**. Tokens de cor são usados consistentemente via CSS variables. Border radius consistente (rounded-xl para cards, rounded-lg para botões).

### Refinamento
⚠️ **Médio**. Sombras são sutis (bom), borders são consistentes, mas falta polish em detalhes: hover states mais ricos, transitions mais suaves, feedback visual mais forte.

### Sensação premium
⚠️ **Média**. Tem potencial (paleta warm, teal usado com moderação, sombras sutis), mas componentes ainda parecem defaults. Falta aquele 20% final de polish que transforma bom em premium.

### Aderência ao alvo Apple-like
⚠️ **Parcial**. Clarity: OK. Deference: OK (UI não compete com conteúdo). Depth: OK (sombras sutis). Quiet confidence: **falta** — ainda parece template, não produto confiante.
